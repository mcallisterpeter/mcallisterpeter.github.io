Site Purpose:

The Weather and Beyond community website is unique in that it not only provides weather informtion, but it does it in conjunction with community activities and events. It will contain a calendar of upcoming events and activities. Community members will have the ability to add events to a calendar. While doing so, weather informtion will be provided so that the best day for the event can be planned. It will also alow community members that wish to attend an activity to anticipate the weather so they know how best to dress.
